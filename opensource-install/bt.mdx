## 新建站点

### 分配域名

写本篇教程的话我准备了一个主域名 `meedu.vip` ，然后我分配了四个子域名用来安装 MeEdu 系统，分配方案如下：

| 服务 | 域名 |
| --- | --- |
| API 服务 | `api.meedu.vip` |
| PC 端口 | `edu.meedu.vip` |
| H5 端口 | `m.meedu.vip` |
| 后台端口 | `admin.meedu.vip` |

您可以按照我的分配方案去做，也可以自行决定子域名的分配，这个没有强制的要求。
