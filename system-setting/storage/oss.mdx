## 购买资源包

浏览器打开 [https://common-buy.aliyun.com/?commodityCode=ossbag](https://common-buy.aliyun.com/?commodityCode=ossbag) 购买OSS资源包，需要购买的资源包如下：

- 标准-同城冗余存储 => 中国内地通用
- 下行流量 => 中国内地通用
- 请求-标准存储类型 => 中国内地通用
